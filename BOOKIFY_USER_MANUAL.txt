1. INTRODUCTION
================================================================================

1.1 WHAT IS BOOKIFY?
-------------------
Bookify is a comprehensive online marketplace platform that connects guests 
with hosts offering accommodations (Homes), experiences, and services. The 
platform facilitates bookings, payments, communication, and management of 
all aspects of the rental and service booking process.

1.2 SYSTEM OVERVIEW
------------------
Bookify is a web-based application that supports three main user types:
- Guests: Users who browse and book listings
- Hosts: Users who create and manage listings
- Admins: System administrators who oversee the platform

The system includes features for:
- Listing creation and management
- Booking and reservation management
- Payment processing (PayPal and wallet system)
- Real-time messaging
- Calendar and availability management
- Points and rewards system
- Wallet and transaction management
- Review and rating system

1.3 USER ROLES
-------------
GUEST ROLE:
- Browse and search listings
- Make bookings
- Manage bookings and cancellations
- Communicate with hosts
- Manage wallet and payments
- Earn and redeem points
- Save favorites
- View and update profile

HOST ROLE:
- Create and manage listings (Homes, Experiences, Services)
- Manage bookings and reservations
- Set availability and pricing
- Process refunds and cancellations
- Communicate with guests
- View earnings and wallet
- Manage calendar
- Update host profile

ADMIN ROLE:
- View system-wide statistics
- Manage users (guests and hosts)
- Manage all listings
- View and manage all bookings
- Monitor transactions and wallet
- Manage rewards system
- Generate reports
- Export data (CSV, PDF, Print)

2. GETTING STARTED
================================================================================

2.1 CREATING AN ACCOUNT
-----------------------
To create a new Bookify account:

1. Navigate to the Bookify homepage
2. Click on "Sign Up" or "Create Account"
3. Fill in the registration form:
   - First Name
   - Last Name
   - Email Address
   - Password (must meet security requirements)
   - Confirm Password
4. Read and accept the Terms of Service and Privacy Policy
5. Click "Sign Up"

ALTERNATIVE: Sign up with Google
- Click "Continue with Google"
- Select your Google account
- Grant necessary permissions

NOTE: New users receive 150 bonus points upon successful registration!

2.2 LOGGING IN
-------------
To log in to your account:

1. Navigate to the Bookify homepage
2. Enter your email address
3. Enter your password
4. Click "Log In"

ALTERNATIVE: Log in with Google
- Click "Continue with Google"
- Select your Google account

FORGOT PASSWORD:
- Click "Forgot Password?" link
- Enter your email address
- Check your email for password reset instructions

2.3 EMAIL VERIFICATION
----------------------
After creating an account:

1. Check your email inbox for a verification email from Bookify
2. Click the verification link in the email
3. You will be redirected to the verification page
4. Once verified, you can access all features

NOTE: Some features may be limited until email verification is complete.

2.4 FIRST-TIME SETUP
--------------------
GUESTS:
- Complete email verification
- Set up your profile (optional but recommended)
- Add payment method to wallet (optional)
- Start exploring listings!

HOSTS:
- Complete email verification
- Navigate to Host Setup
- Complete the host registration process:
  a. Host Setup (Basic Information)
  b. Host Setup 2 (Experiences - if applicable)
  c. Host Setup 3 (Services - if applicable)
- Create your first listing
- Set availability and pricing

3.3 MAKING BOOKINGS
------------------
BOOKING A HOME:
1. Navigate to a home listing
2. Select check-in and check-out dates using the calendar
3. Select number of guests

   - Nightly rate
   - Number of nights

   - Service fee
   - Discounts (if applicable)

5. Apply promo codes or rewards (optional)
6. Click "Reserve" or "Book Now"
7. Review booking details
8. Select payment method:
   - PayPal
   - Wallet balance
9. Complete payment
10. Receive booking confirmation

BOOKING AN EXPERIENCE:
1. Navigate to an experience listing
2. Select a date and time from available schedules
3. Select number of participants
4. Review pricing breakdown
5. Apply promo codes or rewards (optional)
6. Click "Book Now"
7. Complete payment
8. Receive booking confirmation

BOOKING A SERVICE:
1. Navigate to a service listing
2. Select a date and time from available schedules
3. Review pricing
4. Apply promo codes or rewards (optional)
5. Click "Book Now"
6. Complete payment

DISCOUNTS AND PROMOTIONS:

- Coupon codes: Enter code in the booking form
- Rewards: Select from available rewards in your account
- Multiple discounts can stack (promo + coupon + reward)

6. COMMON TASKS & WORKFLOWS
================================================================================

6.1 BOOKING A LISTING
---------------------
COMPLETE WORKFLOW:
1. Guest browses listings
2. Guest finds desired listing
3. Guest selects dates/times
4. Guest reviews pricing
5. Guest applies discounts (optional)
6. Guest clicks "Book Now"
7. Guest selects payment method
8. Guest completes payment
9. Booking confirmation sent
10. Host receives booking notification
11. Host confirms booking
12. Guest receives confirmation
13. Both parties can message each other

6.2 CANCELLING A BOOKING
------------------------
GUEST CANCELLATION:
1. Guest goes to "Bookings"
2. Guest finds booking to cancel
3. Guest clicks "Cancel Booking"
4. System checks cancellation policy
5. If eligible, refund is calculated
6. Guest confirms cancellation
7. Refund processed to guest wallet
8. Host notified
9. Booking status updated

HOST CANCELLATION:
1. Host goes to "Bookings"
2. Host finds booking to cancel
3. Host clicks "Cancel Booking"
4. Host confirms cancellation
5. Full refund processed to guest
6. Guest notified
7. Booking status updated

6.3 PROCESSING REFUNDS
----------------------
HOST REFUND PROCESS:
1. Host goes to booking details
2. Host clicks "Process Refund"
3. Host enters refund percentage (0-100%)
4. System calculates refund amount
5. Host reviews refund details
6. Host confirms refund
7. System checks host wallet balance
8. If sufficient, refund deducted from host wallet
9. Refund added to guest wallet
10. Email confirmation sent
11. Transaction recorded

6.4 MANAGING AVAILABILITY
--------------------------
SETTING AVAILABILITY:
1. Host goes to Calendar
2. Host selects dates
3. Host marks as available/unavailable
4. Host sets pricing for specific dates
5. Host blocks dates for maintenance
6. Changes saved automatically

UPDATING LISTING AVAILABILITY:
1. Host goes to Listings
2. Host edits listing
3. Host updates availability calendar
4. Host saves changes

6.5 APPLYING DISCOUNTS
---------------------
LISTING DISCOUNTS:
1. Host edits listing
2. Host sets discount:
   - Percentage discount
   - Fixed amount discount
3. Host sets discount conditions
4. Host saves listing
5. Discount automatically applied to bookings

PROMO CODES:
1. Admin creates promo code
2. Promo code activated
3. Guest enters code during booking
4. System validates code
5. Discount applied if valid

REWARDS:
1. Guest earns points
2. Guest redeems reward
3. Guest selects reward during booking
4. Discount applied automatically

7.1 COMMON ISSUES
-----------------
LOGIN PROBLEMS:
- Issue: Cannot log in
- Solution: 
  * Check email and password
  * Use "Forgot Password" to reset
  * Clear browser cache
  * Try different browser

PAYMENT ISSUES:
- Issue: Payment not processing
- Solution:
  * Check payment method
  * Verify sufficient funds
  * Check internet connection
  * Try different payment method
  * Contact support

BOOKING ISSUES:
- Issue: Cannot complete booking
- Solution:
  * Check dates are available
  * Verify guest count
  * Check payment method
  * Clear browser cache
  * Try again

MESSAGING ISSUES:
- Issue: Messages not sending
- Solution:
  * Check internet connection
  * Refresh page
  * Clear browser cache
  * Try again

WALLET ISSUES:
- Issue: Funds not appearing
- Solution:
  * Check transaction history
  * Verify payment completed
  * Wait a few minutes
  * Contact support

LISTING ISSUES:
- Issue: Listing not appearing
- Solution:
  * Check listing status (Published)
  * Verify all required fields filled
  * Check filters
  * Contact support
